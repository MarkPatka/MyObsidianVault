|             ∀             | `\forall`         | Квантор всеобщности (для любого)                  |

|             ⊕             | `\oplus`          | Сложение по модулю 2 (исключающее или)            |
|             ⇒             | `\Rightarrow`     | Импликация (следует, если... то)                  |
|             ⇔             | `\Leftrightarrow` | Эквивалентность (тогда и только тогда)            |
|             →             | `\to`             | Стрелка (функция, предел, импликация)             |
|             ⊢             | `\vdash`          | Выводимость (влечет)                              |
|             ⊨             | `\models`         | Логическое следование (общезначимость)            |
|             ⊤             | `\top`            | Истина (тождественно истинно)                     |
|             ⊥             | `\bot`            | Ложь (противоречие)                               |

|             ∖             | `\setminus`       | Разность множеств (без)                           |

|             ≃             | `\simeq`          | Асимптотически равно, гомеоморфно                 |
|             ≡             | `\equiv`          | Тождественно равно, сравнение по модулю           |

|             ∝             | `\propto`         | Пропорционально                                   |
